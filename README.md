# Chiamaka Oluchieze Portfolio Website

Professional static portfolio website for Chiamaka Oluchieze.

## Included
- Responsive frontend built with HTML, CSS and JavaScript
- Profile picture in the hero section
- Uploaded video placed in the Video Work portfolio section
- CV downloads in PDF and Word formats
- Recruiter-focused sections for About, Portfolio, Expertise, Experience and Contact
- Vercel-ready configuration

## Deploy to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Open this folder in a terminal.
3. Run `vercel login` and follow the prompts.
4. Run `vercel --prod` to publish the website.
5. Copy the live deployment URL from the terminal.

## Add more portfolio files
Place additional videos in `assets/videos/` and graphics/images in `assets/images/`, then add new portfolio cards in `index.html` inside the `portfolio-grid` section.
